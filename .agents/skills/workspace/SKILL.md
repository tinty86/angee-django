# Workspace

Use the `angee-workspace` skill in workspace mode.

For workspace creation, derive the parent ref using the Angee workspace rules and
create the dev workspace with `angee ws create`. For status, inspect Angee's
native workspace and GitOps state.

Load `.agents/skills/angee-workspace/SKILL.md` and follow its Create Workspace
and reporting workflows.
